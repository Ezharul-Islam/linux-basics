## Week 2 – Users, Groups & Permissions

In Week 2, I learned how Linux controls access and security.

### Topics Covered
- Linux users and groups
- Creating users and groups
- File permissions (r, w, x)
- chmod (permission control)
- File ownership and chown
- sudo and administrative access
- Real permission scenarios and practice

### Key Learnings
- Ownership decides who controls a file
- Permissions decide what actions are allowed
- Only owner or root can change permissions
- sudo gives temporary admin power
- Linux security works automatically and strictly

### Learning Method
- Hands-on practice
- Real-life (hotel) analogy for better understanding
- Step-by-step observation using ls -l
